# Spoitfy Clone

An Spotify app made with Electron Typescript Vite using SpotifyAPI. you can login there and you can see your playlist and play songs..

## Project Setup

### Install

```bash
$ npm install
```

### Development

```bash
$ npm run dev
```

### Build

```bash
# For windows
$ npm run dist:win

# For macOS
$ npm run dist:mac

# For Linux
$ npm run dist:linux
```
